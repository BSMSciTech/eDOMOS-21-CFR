## üîç TROUBLESHOOTING

### Common Issues and Solutions

**Issue**: PDF generates but shows blank pages
- **Solution**: Check ReportLab library version (need 3.6+)
- **Solution**: Verify event data exists for selected date range

**Issue**: Colors don't print correctly
- **Solution**: Enable color printing in printer settings
- **Solution**: Use color-capable printer (not B&W only)

**Issue**: Table spans multiple pages awkwardly
- **Solution**: This is normal; headers repeat automatically
- **Solution**: Adjust date range to control report length

**Issue**: Fonts look pixelated
- **Solution**: Print at higher DPI (minimum 600 DPI)
- **Solution**: Use PDF viewer's "Print as Image" option

**Issue**: Signature lines not visible
- **Solution**: Ensure certification section renders properly
- **Solution**: Check if printer cuts off bottom margin

**Issue**: Document ID not unique
- **Solution**: Timestamp-based ID ensures uniqueness down to the second
- **Solution**: If generating multiple in same second, wait 1-2 seconds
